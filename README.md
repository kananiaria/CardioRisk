# CardioRisk
Its a ML-based cardiovascular risk stratification tool built using Logistic Regression and Random Forest, providing probabilistic risk scores and feature-based explanations to support early screening and referral decisions.
Built using:
- React
- Vite
- Tailwind CSS
- Flask
- Logistic Regression
- Scikit-learn
- Cleveland Heart Disease Dataset
Features are:
- Professional healthcare dashboard
- Explainable risk predictions
- Low, Moderate, and High risk categorization
- Primary and secondary contributors
- Suggested actions
